# Frontend (Auction Web Client)

## Purpose
The frontend provides the **user interface** for the Auction System.

It allows users to register, advertise items, and participate in real-time bidding.

---

## Features
- User authentication
- Auction listings
- Live bid updates
- Countdown timers
- Winner notifications

---

## Technologies
- Web-based UI
- REST API integration
- WebSocket connection for real-time updates

---

## Notes
The frontend is deployed as part of the SaaS offering and communicates with backend services via the API Gateway.
